CONFIGURING A SWITCH
3. Security configuration of the switch
ena
conf t
host S0
line con 0
pass cisco
login
line vty 0 15
pass ciscoexam
login
exit
ena sec ciscoexam12345
banner motd # Authorized Access Only!!! #
ser pass
do wr

4. Create vlans: vlan 10, vlan 20, vlan 30.
##For Net1:
S0(config)#int range fa0/2-4
S0(config-if-range)#switchport mode access
S0(config-if-range)#switchport access vlan 10
% Access VLAN does not exist. Creating vlan 10

##For Net2:
S0(config)#int range fa0/5-7
S0(config-if-range)#switchport mode access
S0(config-if-range)#switchport access vlan 20
% Access VLAN does not exist. Creating vlan 20

##For Net3:
S0(config)#int range fa0/8-10
S0(config-if-range)#switchport mode access
S0(config-if-range)#switchport access vlan 30
% Access VLAN does not exist. Creating vlan 30
S0(config-if-range)#exit
S0(config)do sh start ##to see if the vlan is created

6. Trunking on a switch port  
int fa0/1
switchport mode trunk 
## This command sets the mode of the specified switch port to trunk. This tells the switch to allow traffic from all VLANs to be sent and received on this port, with each frame tagged to identify its source VLAN.
do wr

3. Config intervlan 
## gagawa ng sub-interfaces sa interface ng router (g0/0). Here, ilalagay natin ang ip add ng sub-interface na ito or sabihin natin sa vlan na ito.

R1(config)#int g0/0.10 ##This creates a subinterface for VLAN 30. A subinterface is a virtual interface created on top of a physical one, allowing it to handle traffic for a specific VLAN.
R1(config-subif)#encapsulation dot1Q 10 ##tells the router to use the 802.1Q VLAN tagging standard.
R1(config-subif)#ip address 192.168.1.1 255.255.255.192
R1(config-subif)#do wr
R1(config-subif)#exit

R1(config)#int g0/0.20
R1(config-subif)#encapsulation dot1Q 20
R1(config-subif)#ip address 192.168.1.1 65.255.255.192
R1(config-subif)#do wr
R1(config-subif)#exit

R1(config)#int g0/0.30
R1(config-subif)#encapsulation dot1Q 30
R1(config-subif)#ip address 192.168.1.129 65.255.255.192
R1(config-subif)#do wr
R1(config-subif)#exit
R1(config)#do sh start ##to see if naconfigure talaga sya

4. Configure the DHCP server of the router.
R1(config)#service dhcp
R1(config)#ip dhcp pool admin-pool
R1(dhcp-config)#net 192.168.1.0 255.255.255.192
R1(dhcp-config)#default-router 192.168.1.1
R1(dhcp-config)#dns-server 192.168.1.1
R1(dhcp-config)#domain-name admin.com
R1(dhcp-config)#exit
