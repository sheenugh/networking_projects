IX. CONFIGURING ACCESS POINTS
Go to config to port 1. Change the SSID and put a password by clicking first the WPA2-PSK. Must be AES (below the password).


X. CONNECTING THE WIRELESS DEVICES
Try the wireless connection. Connect the wireless devices by adding smartphone, laptop etc.
a. Smartphone	
Config Category
Click wireless0. Input the wifi SSID and password (WPA2-PSK)
Desktop Cate > IP Config
click the dhcp for its ip

b. Laptop
Physical Category
Off the laptop
Remove the current module. Add the module WPC300N. Then, turn on.
Config Category
Click wireless0. Input the wifi SSID and password (WPA2-PSK)
OR
Desktop Category > IP Wireless
Refresh. Connect to the designated wifi. Input the password. Connect.
Desktop Cat > IP Config
click the dhcp for its ip

c. Tablet
Config Category
Click wireless0. Input the wifi SSID and password (WPA2-PSK)
Desktop Cate > IP Config
click the dhcp for its ip


XI. CONFIGURE SSH
ip domain-name cisco.com
crypto key generate rsa
1024 ##(Press Enter and then choose a key size like 1024 or 2048)
username <your-username> secret <your-password>
line vty 0 15
no password
no login
transport input ssh
login local
end
